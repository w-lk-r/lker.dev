# Link Preview Images

Add static preview images here as fallbacks for the link cards.

## Required Images:

- `github.png` - Preview image for GitHub profile
- `linkedin.png` - Preview image for LinkedIn profile
- `icodethis.png` - Preview image for iCodeThis profile

## Recommended Size:

- Width: 1200px
- Height: 630px (or maintain a 1.91:1 aspect ratio)

These images will be used as fallbacks when Open Graph scraping fails or is blocked.
